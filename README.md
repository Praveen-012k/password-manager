# Password Manager

A desktop password manager application with random password generation and secure storage. Built with Python and Tkinter.

## Features

- Generate strong random passwords (8-18 characters with letters, numbers, symbols)
- Save website credentials to JSON file
- Search functionality to retrieve saved passwords
- Auto-copy generated passwords to clipboard
- Input validation and error handling

## Technologies Used

- Python
- Tkinter (GUI)
- JSON (Data storage)
- Pyperclip (Clipboard management)

## How to Run

1. Clone this repository
2. Install required packages: `pip install pyperclip`
3. Run: `python main.py`

## Security Note

This is a learning project. For production use, consider encrypting stored passwords.
